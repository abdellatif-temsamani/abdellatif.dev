# Development

## Requirements

- [Node.js](https://nodejs.org/en/)
- [pnpm](https://pnpm.io/)

IDE of your choice, I recommend [neovim](https://neovim.io/) with the
[lsp-config](https://github.com/neovim/nvim-lspconfig) plugin.

## start the development server

```bash
pnpm dev
```

## build the project

```bash
pnpm build
```
